1

Configurar Meta for Developers

WhatsApp Business API + Facebook Messenger API — Gratis para empezar
1

Crear cuenta en Meta for Developers

Andá a developers.facebook.com y creá una cuenta de desarrollador con tu usuario de Facebook (el mismo que usás para tu Marketplace).

Ir a Meta Developers →
2

Crear una App de tipo "Business"

En el panel, hacé clic en Create App → elegí tipo Business → dale un nombre (ej: "MoshiClaw Bot"). Esto crea el contenedor para todas las APIs.

3

Activar WhatsApp Business API

  • Dentro de tu App, buscá el producto WhatsApp y hacé clic en Set Up
  • Creá o vinculá tu WhatsApp Business Account (WABA)
  • Agregá tu número de WhatsApp Business para verificarlo
  • Copiá el Phone Number ID y el Access Token permanente — los vas a necesitar para n8n
📱 La API de WhatsApp es gratuita para las primeras 1.000 conversaciones por mes iniciadas por clientes. Perfecto para un negocio en Marketplace.
4

Activar Messenger API (para Facebook Marketplace)

  • En tu App, agregá el producto Messenger
  • Conectá tu Página de Facebook (la del negocio en Marketplace)
  • Generá un Page Access Token permanente
  • Anotá el Page ID de tu página
ℹ️ Si tu negocio en Marketplace está asociado a tu perfil personal (no a una Página), vas a necesitar crear una Página de Facebook para el negocio — es gratuito y muy sencillo.
5

Guardar las credenciales

Anotá en un lugar seguro:

  • ✅ WhatsApp: Phone Number ID + Permanent Access Token + WABA ID
  • ✅ Messenger: Page Access Token + Page ID + App Secret
✅ Con esto tenés las credenciales necesarias. El siguiente paso es configurar n8n para usarlas.
2

Configurar n8n

El motor de automatización que conecta todo

☁️ Opción A: n8n Cloud Recomendada

Registrarse en n8n.io. Tiene prueba gratuita de 14 días, luego desde ~$20/mes. No requiere instalar nada. Ideal para empezar rápido.

🖥 Opción B: n8n Self-hosted Gratis

Instalar en tu propia computadora o servidor. Completamente gratuito. Requiere que la PC esté encendida para que funcione. Ideal si querés control total.

1

Conectar n8n con MoshiClaw

Una vez que tengas tu instancia de n8n, necesitás conectarla a MoshiClaw. Para eso:

  • En n8n, andá a Settings → API → Create API Key
  • Copiá la URL de tu instancia (ej: https://tuapp.app.n8n.cloud) y la API Key
  • Decile a MoshiClaw: "Conectá mi n8n con esta URL y esta API key"
💡 Después de conectar n8n, MoshiClaw puede activar, pausar y ejecutar workflows directamente desde el chat.
2

Importar los workflows de MoshiClaw

MoshiClaw te va a crear y subir automáticamente los workflows necesarios una vez que esté conectado a n8n. Los workflows incluyen:

  • WhatsApp Receiver: escucha mensajes entrantes de WhatsApp
  • Messenger Receiver: escucha mensajes de tu página de Facebook
  • Stock Checker: lee tu Excel/Sheets para verificar disponibilidad
  • AI Responder: genera la respuesta usando Claude IA
  • Mode Controller: decide si responder auto o pedir aprobación
3

Configurar las credenciales en n8n

En n8n, ir a Credentials → Add Credential y agregar:

  • WhatsApp Business Cloud API → pegar el Access Token y Phone Number ID
  • Facebook Graph API → pegar el Page Access Token
  • Anthropic API → pegar tu API key de Claude (o usar la integración nativa)
  • Google Sheets (opcional) → para leer el stock en tiempo real
3

Configurar el Stock

Ya tenés el Excel listo — también funciona con Google Sheets
1

El archivo Excel ya está creado

Encontrás stock_moshiclaw.xlsx en tu carpeta. Tiene 3 hojas:

  • STOCK: tus productos, precios, stock actual, y configuración por producto
  • PARÁMETROS IA: comportamiento global del bot (modo, mensajes, horarios, bloqueos)
  • REGISTRO MENSAJES: historial de todos los mensajes gestionados por la IA
2

Columnas clave en la hoja STOCK

  • RESPONDER MENSAJES: SÍ/NO — si la IA responde consultas sobre ese producto
  • MODO RESPUESTA: AUTO (envía sola) / SEMI (te pide aprobación) / PAUSADO
  • PRECIO MÍNIMO: la IA nunca va a comprometer un precio por debajo de este valor
  • DESCRIPCIÓN (para la IA): texto que la IA usa para describir el producto al cliente
💡 Si el stock de un producto llega a 0, la IA automáticamente deja de responder sobre ese artículo (o avisa que no hay, según la configuración).
3

Opción Google Sheets (actualización en tiempo real)

Si subís el Excel a Google Sheets, n8n puede leer los datos en tiempo real sin que tengas que subir archivos. Ventajas:

  • Editás el stock desde el teléfono y la IA lo ve al instante
  • Podés compartirlo con empleados para que actualicen precios
  • MoshiClaw puede actualizar el stock directamente cuando lo pedís
4

Controlar todo desde MoshiClaw

Una vez configurado, todo se maneja desde tu chat
1

Comandos que podés darle a MoshiClaw

  • 🟢 "Activá el auto-responder de WhatsApp"
  • 🔴 "Pausá todo por hoy"
  • 🚫 "No le respondas más al número +54 11 XXXX XXXX"
  • 📦 "Actualizá el stock del iPhone 14 a 5 unidades"
  • 💰 "Cambiá el precio del Galaxy S23 a $580.000"
  • 📋 "Mostrame los mensajes que están esperando aprobación"
  • ✅ "Aprobá todas las respuestas pendientes"
  • ⚙️ "Cambiá el modo de los AirPods a AUTO"
  • 📊 "¿Cuántos mensajes respondió la IA hoy?"
2

Parámetros configurables en tiempo real

  • Modo global: AUTO / SEMI / PAUSADO — para todo el sistema
  • Por producto: cada artículo tiene su propio modo
  • Lista de bloqueos: números o nombres que la IA ignora
  • Reglas de precio: umbrales mínimos y comportamiento ante ofertas
  • Horario de atención: mensajes automáticos fuera de horario
✅ Todo esto se configura hablando con MoshiClaw. No necesitás tocar código ni entrar a n8n cada vez.
